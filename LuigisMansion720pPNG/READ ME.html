Installation:

To install the Luigi's Mansion HD Texture Pack, just drop the folder called "GLM" into the "Load" folder in your Dolphin directory. In Ishiiruka Dolphin, make sure that "Load Custom Textures" and "Load custom material maps" is enabled in "Graphics" >> "Advanced". Now run the game in Ishiiruka Dolphin (not normal Dolphin!). If you encounter any more issues please contact me on the texture pack download page or on my Discord server

~Henriko

Are you experiencing heavy frame drops and lag spikes while using my texture pack? Please enable the option "prefetch custom textures" in Dolphin, if that doesn't work upgrade the amount of RAM in your system to solve this issue.
